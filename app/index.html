Octohedron

You cannot see anything around you in the pitch black chamber. You can only rely on your trusty HUD (Heads-Up-Display) built in to your helmet visor. However that oh-so-trusty visor display malfunctioned and lost its map of the maze, all you can see is just the information on the current chamber you are in. Scrum*tious

SCRUM*TIOUS `is a virtual scrum group where users can hold remote stand-ups. This CRUD application is built using NodeJS/Express back end framework with MongoDB as the document store. My role in this project was the Version Control Manager; I was responsible for maintaining the remote repository on Github, testing and validating pull request from my team. On the back-end, I was responsible for building out the endpoints for consumption; and on the front-end, I handled Lodash templating and some styling with Materialize CSS.

Ondex

ONDEX is an application where users can post an item or service up for sale, trade, auction, or barter. Ondex was built using Ruby on Rails with PostgreSQL handling database duties. It features a hand-rolled user authentication system, file attachments with Paperclip ruby gem and Amazon's S3 service for content delivery. MaterializeCSS framework for styling elements. Deployment on the Heroku application platform.

Humans of Panda

HUMANS OF PANDA is a Proof-of-Concept submission for Panda Restaurant Group Inc. This application serves as an internal social-network for the restaurant, catering, and sales department employees to be able to connect, share, and acknowledge peer activity and accomplishments. This Single-Page-Application was built using AngularJS client-side, backed by NodeJS Express Framework, and MongoDB serving data.
